# Deprecated Standards

HD 131156 = Xi Boo A = HR 5544 = GJ 566 (J1451+1906)
  G8V: MK43(stan),JM53,Wilson62(stan,JM53),Houk,Abt81
  G7V: Levato78(G7Ve),Gray03
  G5: Bidelman85

  B-V=0.733+-0.005 (CCDM Tycho=>Johnson photometry, EEM). As Gray's
  more recent classifications are more reliant on Keenan's 1980s-era
  standards, it appears that Gray03 considers Xi Boo A to be ~1
  subtype earlier than 61 UMa. As 61 UMa is an anchor standard that
  has survived from MK1943 through Keenan's 1980s works, we keep 61
  UMa as the primary G8V standard, and now consider Xi Boo A to be
  deprecated.

# Variant

HD 82885 = 11 LMi
  G8Va: Keenan99
  G8V: Abt81
  G8IV: Abt08
  G8IV-V: Roman52

  B-V=0.770, V=5.40 (HIP). Good agreement among classifiers on "G8"
  temperature type, but mild disagreement on luminosity class between
  V, Va, IV, IV-V. Using revised Hipparcos parallax, I find Mv = 5.12,
  so for its B-V it is 0.39 mag above the main sequence, so it might
  be best considered "G8IV-V"?
