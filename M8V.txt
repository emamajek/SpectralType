# Other Standard

2MASS J1047+4026 [= LP 213-68 = LDS 4016B = 2MASS J10471381+4026493]

  Some sleuthing is required to figure out which star "2MASS
  1047+4026" is - this system is a poster-child for the foolishness
  that is rampant abbreviation of 2MASS/SDSS phone number names.
  Triple system: "A" (LP 213-67) is tight binary with separation
  0.122" (CLO 5; mag=12.85, 13.7) composed of M8+L0 binary
  (Close03). "B" (LP 213-68) is 14.5" away. "AB" comprise LDS
  4016. V=17.36(SIMBAD, no ref.), J=11.384+-0.022(2MASS),
  H=10.777+-0.019(2MASS), Ks=10.399+-0.018(2MASS). So which is the M8V
  "standard"? Gizis00 lists "A" as M6.5Ve and "B" as M8Ve (Reid02
  agrees), however Close03 calls "A" M8+L0. So my bet is that
  Hawley02's "2MASS J1047+4026" is LP 213-68, so that is what I have
  assumed for the aliases listed above.
